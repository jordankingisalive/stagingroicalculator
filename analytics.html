⚠️ Analytics Setup Required

To view analytics data, you need to replace YOUR_CLARITY_PROJECT_ID with your actual Microsoft Clarity Project ID.

Quick Setup:

  1. Go to Microsoft Clarity and sign in
  2. Create a new project for your website
  3. Copy the Project ID from your dashboard
  4. Replace YOUR_CLARITY_PROJECT_ID in all HTML files
  5. Wait 15-30 minutes for data to start appearing
