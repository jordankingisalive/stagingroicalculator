Getting Started

1

Export Your Heatmap Data

Export data from the Copilot Super Usage Report heatmap. Get the report at https://aka.ms/decodingsuperusage. Choose your export method below:

▶ 💻 Export from Power BI Desktop
  1. Open the .pbit file in Power BI Desktop
  2. Ctrl+Click ALL options on the left-hand side of the heatmap
  3. Click on the heatmap visualization
  4. Click the More options (⋯) menu at the top right of the visual
  5. Click "Export data"
  6. Save the CSV file
▶ 🌐 Export from Power BI Web (app.powerbi.com)
⚠️ Note: Exporting visual data from the Power BI web app requires your tenant admin to have enabled the "Export data" setting. If the option below is not available, use Power BI Desktop instead.
  1. Open the report on app.powerbi.com
  2. Ctrl+Click ALL metric options on the left-hand side of the heatmap (Enabled Users, % Active Users, etc.)
  3. Click on the heatmap table to select it
  4. Hover over the visual — look for a small ⋯ icon in the top-right corner of the visual's header bar
  5. If the icon appears, click it and select "Export data", then choose "Summarized data" → ".csv" → Export
  6. If no icon appears, your organization's admin settings may prevent visual-level data export — use Power BI Desktop instead

Tip: The top menu bar "Export" button only exports the full report as PDF, PowerPoint, or an Excel PivotTable — it does not export the visual's underlying data as a flat CSV.

Optional: Rename the downloaded file to heatmap.csv for easier identification.

Expected columns: Team/Division Name, Active Users, Enabled Users, Total Actions, Monthly Actions, Engagement %, etc.

📊 Intelligent Recap Usage Guidance

Microsoft does not publish an industry-wide benchmark for Intelligent Recap actions per 1,000 employees. However, observed usage shows active users average ~1.6 Intelligent Recap actions per week. When normalized by adoption rate, this typically equates to ~700–2,000 recap actions per 1,000 employees per month, depending on enablement and meeting load.
